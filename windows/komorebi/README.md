# Komorebi

## Files Included
- `whkdrc` - keybinds for komorebi
- `komorebi.json` - configurations for komorebi
- `komorebi-run.ps1` - a script to run komorebi
- `komorebi-stop.ps1`- a script to terminate komorebi 
- `komorebi-nobar.ps1` - a script to run komorebi without komorebi-bar
 
## How to Use
Visit https://github.com/LGUG2Z/komorebi for more information on how to use these configs.

In short, these files should be put at:
- whkdrc: "C:/Users/{username}/.config/"
- komorebi.json "C:/Users/{username}/"

Where C: is the drive with your Windows installation.

As for the powershell scripts, you can put them anywhere you want but i saved them in the start menu so i can search ffor them.
